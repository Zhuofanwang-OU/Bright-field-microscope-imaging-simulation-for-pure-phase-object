# Bright-field microscope imaging simulation for pure phase spherical particles

This repository contains physics-based bright-field microscope simulations for
colloidal-scale spherical particles / beads treated as pure phase objects.

The current version replaces the old `Partial coherent+Transfer+pupil(No Mie).py`
script with updated partial-coherence forward models. The simulation can generate
through-focus defocus image stacks for micron-scale spherical particles, and the
particle size and microscope parameters are adjustable.

## Main scripts

- `Partial coherent+Transfer+pupil(OPD).py`  
  Partial coherent bright-field simulation using a pure phase optical path
  difference (OPD) sphere. The object is modeled as
  `exp(i * 2*pi/lambda * (n_bead - n_medium) * thickness)`.

- `Partial coherent+Transfer+pupil(Mie).py`  
  Partial coherent bright-field simulation using a scalar Mie angular-spectrum
  object field. This version includes complex scattering amplitude and phase
  structure for spherical particles. The absolute scattering strength is
  controlled by the adjustable `scattering_scale` parameter.

- `compare_opd_experiment_ssim.py` and
  `compare_opd_experiment_contrast_masked.py`  
  Example analysis scripts for comparing simulated and experimental
  through-focus particle images.

## Adjustable parameters

The simulation functions expose the main particle and microscope settings:

- particle size: `bead_radius_um` or `bead_diameter_um`
- refractive indices: `n_bead`, `n_medium`
- objective magnification: `magnification`
- objective NA: `NA_obj`
- illumination NA: `illumination_NA`
- source sampling: `source_sigma`, `n_src`
- camera pixel size: `sensor_pixel_size_um`
- defocus planes: `z_list`
- simulation grid: `grid_size`, `pad_factor`
- Mie coupling strength: `scattering_scale`

These parameters allow the same code to simulate different colloidal-scale
spherical particles and different microscope configurations.

## Example usage

```python
from importlib.machinery import SourceFileLoader

opd = SourceFileLoader(
    "opd_model",
    "Partial coherent+Transfer+pupil(OPD).py",
).load_module()

opd.simulate_partial_coherent_opd_imaging(
    save_dir="results_custom_opd",
    magnification=20.0,
    NA_obj=0.40,
    illumination_NA=0.09,
    bead_radius_um=1.0,
    n_bead=1.59,
    n_medium=1.50,
    z_list=list(range(-50, 51, 10)),
    show_plot=False,
)
```

```python
from importlib.machinery import SourceFileLoader

mie = SourceFileLoader(
    "mie_model",
    "Partial coherent+Transfer+pupil(Mie).py",
).load_module()

mie.simulate_partial_coherent_mie_imaging(
    save_dir="results_custom_mie",
    magnification=40.0,
    NA_obj=0.60,
    illumination_NA=0.09,
    bead_diameter_um=2.0,
    n_bead=1.59,
    n_medium=1.50,
    scattering_scale=0.18,
    z_list=list(range(-50, 51, 10)),
    show_plot=False,
)
```

## Example outputs

The included `results_tif_*` folders contain example through-focus TIFF stacks
and contact sheets generated by the updated models. The TIFF output is raw
intensity without flat-field normalization.
